# First Aid LMS (Admin) – Code Repository

이 레포는 2차 팀 프로젝트 **응급처치 교육 관리자 페이지**의 전체 코드를 포함합니다.  
관리자 대시보드, 회원 관리, 강의 관리, 교육 자료, 문의 답변 기능 등을 직접 구현했습니다.

---

## 주요 기능 요약
- 관리자 대시보드 통계 UI
- 회원 · 강의 · 교육자료 CRUD
- 게시물 등록/수정 폼 검증
- 문의사항 답변 관리
